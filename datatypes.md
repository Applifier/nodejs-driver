# CQL types to javascript types

There are few data types defined in the ECMAScript standard for javascript...


CQL3 data type | javascript type
--- | ---
ascii | String
bigint | Long
blob | Buffer
boolean | Boolean
counter | Long
decimal | not supported
double | Number
float | Number
inet | Buffer
int | Number \*
list | Array
map | Object \*
set | Array
text | String
timestamp | Date
timeuuid | String \*\*
tuple | not supported yet
uuid | String  \*\*
varchar | String
varint | not supported
